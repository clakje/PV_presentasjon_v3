# Scrollytelling: Pedagogisk veileder for Helse Sør-Øst

## Formål
Å presentere Helse Sør-Øst sin pedagogiske veileder på en innovativ, moderne og engasjerende måte via scrollytelling. Målgruppen er ressurssykepleiere ansatt i HSØ. 

## Læringsmål
Ressurssykepleierne skal forstå hvordan den pedagogiske veilederen kan hjelpe dem med å strukturere og effektivisere kompetansehevingen de jobber med til daglig.

## Kjernefunksjoner (MVP)
- Scrollytelling-navigasjon der innhold (tekst og grafikk) beveger seg inn og ut basert på rulling.
- Dynamiske bakgrunner som endrer seg sømløst nedover på siden.
- God og behagelig brukeropplevelse for navigering.
- Design forankret i Nasjonalt profilprogram for helseforetakene (Helse Sør-Øst).

## Teknisk stabel
- Språk: HTML, CSS, JavaScript (Vanilla).
- Format: Skal kunne kjøres 100% lokalt i en nettleser (krever ingen server/backend).

## Det vi IKKE skal bygge (ennå)
- Interaktivt innhold (som quiz eller klikkbare dybdeoppgaver).
- Løsninger som krever database eller server for å fungere.
- Avansert universell utforming (ikke et krav for denne MVP-en).
